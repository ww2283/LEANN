# leann-core

Core APIs, CLI (`leann`), embedding pipeline, and backend plugin registry for [LEANN](https://github.com/yichuan-w/LEANN) — a lightweight local vector/RAG stack.

For installation, backends (HNSW, IVF, DiskANN), and full documentation, see the [main repository README](https://github.com/yichuan-w/LEANN/blob/main/README.md).
